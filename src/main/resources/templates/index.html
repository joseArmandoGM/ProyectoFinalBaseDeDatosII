¡Bienvenido al Sistema de Gestión de Eventos del Campus Jalpa!

Tu portal para una vida universitaria organizada y eficiente.

Este sistema te permite gestionar de manera centralizada todas las actividades académicas, extracurriculares y administrativas. Con una interfaz intuitiva y fácil de usar, podrás:

  • Visualizar las actividades dadas de alta: Accede a una vista general de todas actividades con las fechas y lugares en que se realizan, el cupo de personas, el evento al que pertenecen, etc.
  • Dar de alta actividades: Registra actividades, especificando la información y los detalles de las mismas.
  • Ver información específica de las actividades: Además de poder eliminar o modificar las actividades que ya estén registradas.
  • Consultar las categorias: Consultar las categorías: Revisa una lista con todas las categorías a las que pueden pertenecer las actividades, agrega nuevas o edita y elimina las ya existentes.
  • Buscar actividades o categorías específicas utilizando la barra de búsqueda.
  • Realizar consultas y subconsultas para obtener información específica.
